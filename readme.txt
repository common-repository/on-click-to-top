== Description ==

This plugin can help to add a "Scroll to Top" button in your website bottom. This plugin is a easy to install and no Admin configeration needed. So Just Install and Active. Than just take a look on your or Client website "Bottom-Right" Corner. Thank's for installing. ❤❤❤


### If you found any bug or have a question contact us at freelancerrahim26@gmail.com

== Installation ==

### First download the the plugin ZIP file.  

1. Then go to Plugins page Add New > Upload Plugin .  
2. Click "Choose file" then select our plugin .zip file. 
3. Then Install and activate the plugin.
4. Go to your site any page and scroll down > Take a look on your site bottom-right corner.
